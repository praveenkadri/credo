# Right Rail System — Wealthsimple-Like Support Rail

## Intent
The right rail must be supportive, not competitive.

## Module feel
Support modules should feel:
- soft
- calm
- lighter than the chart
- slightly stronger than plain canvas
- not like stacked dashboard widgets

## Suggested panel classes
- `rounded-[28px] bg-[#fafaf7] p-5`
- optional tiny shadow only

## Header style
- small uppercase label
- quiet section name
- minimal metadata

## Row / content style
- medium-length content
- generous spacing
- no dense lists unless necessary
- subtle hover only when clickable

## Rules
- at most one module can feel semi-prominent
- rest should remain low-contrast
- avoid identical intensity on all modules
- no card-stack feeling
