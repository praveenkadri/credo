# Topbar System — Wealthsimple-Like Utility Band

## Intent
The topbar should feel like a very soft utility shelf.

## Structure
- search
- date / utility controls
- profile / workspace switch

## Suggested classes
Topbar band:
- `rounded-2xl bg-[#f7f7f4] px-5 py-3`
- very soft shadow only if needed

Search:
- `h-11 rounded-xl bg-[#fafaf7] px-4 text-sm text-[#575b55]`

Quiet control:
- `h-11 rounded-xl bg-[#fafaf7] px-4 text-sm font-medium text-[#575b55]`

Icon button:
- `size-11 rounded-xl bg-[#fafaf7] text-[#575b55]`

## Rules
- no dark nav
- no obvious toolbar chrome
- no thick outlines
- controls should look like they rest on the canvas, not float loudly
