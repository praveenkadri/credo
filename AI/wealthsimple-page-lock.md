# Wealthsimple Page Lock — Overview / Portfolio Style Pages

Use this file for any overview page that includes:
- a hero balance
- a money movement graph
- account / company rows underneath
- support modules on the right

## Page structure lock

The page must read in this order:

1. utility shell
2. hero value + delta
3. hero graph
4. lightweight support content
5. account-like rows / lower lists

## Hero section lock

The hero must contain:
- eyebrow label
- strong title or context label
- large value
- delta directly below
- graph with substantial width
- quiet timeline controls

The hero graph should occupy more visual importance than any panel below it.

## Lower section lock

Lower sections must feel like:
- rows
- account summaries
- lightweight tiles
- supportive content

They must not feel like equal-status cards.

## Right rail lock

The right rail should mimic WS support areas:
- smaller
- cleaner
- secondary
- useful but quiet

At least one right-rail module can be slightly stronger, but none should compete with the hero graph.

## Spacing lock

The page needs more whitespace than a normal dashboard.
If it feels dense, it is wrong.
If all modules touch the eye with equal intensity, it is wrong.

## Exact anti-drift rule

Do not let overview pages drift toward:
- B2B dashboard grids
- finance admin cards
- analytics card stacks
- enterprise filter layouts

Return them to:
- calm wealth surface
- balance-first hierarchy
- graph-first storytelling
