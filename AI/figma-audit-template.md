# Figma Audit Template — Wealthsimple Page QA

## Score each 1–10
- Hero hierarchy
- Chart fidelity
- Canvas openness
- Sidebar quietness
- Topbar restraint
- Right-rail support quality
- Lower-row lightness
- Typography precision
- Color discipline
- Overall WS likeness

## Visual checks
- [ ] Hero value clearly anchors the page
- [ ] Delta sits directly under hero value
- [ ] Chart is dominant
- [ ] Chart line is thin and elegant
- [ ] Positive state uses restrained green
- [ ] Right rail is secondary
- [ ] Lower rows feel like account / list content, not cards
- [ ] Controls are small and quiet
- [ ] Canvas feels open
- [ ] No dashboard grid feel

## Fail conditions
- [ ] Generic card dashboard
- [ ] Boxed chart
- [ ] Too much chrome
- [ ] Right rail too heavy
- [ ] Equal weight everywhere
