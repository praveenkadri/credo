# Hero Graph Lock — Exact Behavior Rules

Use when implementing the main graph.

## Must-have behaviors
- large money value above the graph
- positive delta directly below
- graph gets the widest, tallest space in the main column
- active point dot
- vertical hover guide
- pre-hover line strong, post-hover line faded
- soft area fill under active portion
- quiet timeline controls below
- optional Value / Returns switch lower-right

## Must-have composition
- wide horizontal graph
- minimal axis chrome
- graph field open and almost borderless
- no nested chart cards
- no enterprise analytics labels

## If a point is hovered
Display:
- value
- time / period label
- active point marker

The value display can live in the main header or near the hover point, but it must feel precise and calm.

## Final rule
If the graph feels like a widget, rebuild it.
